# 자바스크립트 비트연산자

| 연산자          | 유형            | 설명                  |
|--------------|---------------|---------------------|
| &            | AND           |                     |
| &#124;       | OR            |                     |
| ^            | XOR           |                     |
| ~            | NOT           |                     |
| &lt;&lt;     | 왼쪽 시프트        |                     |
| &gt;&gt;     | 오른쪽 시프트       | 왼쪽은 제일 큰 비트의 값으로 채움 |
| &gt;&gt;&gt; | 부호 없는 오른쪽 시프트 | 왼쪽은 0으로 채움          |
